Danger Zone: Manual Purge

These actions are destructive and cannot be undone. Please proceed with extreme caution.

Purge Inactive Users

Deletes all users (non-admin) who haven't uploaded an image in the last 3 months, or ghost accounts created >3 months ago with no uploads.

Purge All Images

Deletes EVERY image from the server database and storage, regardless of expiration date. Users remain.

Nuclear Purge

Deletes EVERYTHING: All images, all logs, announcements, and ALL users except Owner and Admins. Banned IPs are preserved.
